# مركز مهام التواصل (WMP Outreach Hub)

منصة داخلية مغلقة لشركة Web Marketing Pro لإدارة مهام التواصل مع الحسابات المستهدفة عبر إنستغرام.

## الميزات الأساسية
- **المدير (Admin)**: استيراد جماعي ذكي للحسابات (JSON) مع اكتشاف التكرار، والتوزيع العادل التلقائي على الموظفين الأقل حملاً. لوحة قيادة متكاملة.
- **الموظف (Agent)**: واجهة هاتفية سلسة (Mobile First)، مع مهام متدرجة (Stepper) تضمن نسخ الرسائل وفتح الحسابات وتأكيد المتابعة.
- **نظام النقاط والمتابعة**: يتم احتساب النقاط تلقائياً بناءً على إنجاز المهام وجدولة المتابعات بدقة بعد 3 أيام بشكل آمن.

## التشغيل محلياً

1. الاعتماديات تم تثبيتها مسبقاً عبر `npm install`.

2. قم بتجهيز قاعدة البيانات Supabase:
   - قم بنسخ محتوى الملفات الموجودة في `supabase/migrations/` وتشغيلها بالترتيب (أو عبر `supabase db push`).
   - قم بإنشاء المستخدمين عبر واجهة Supabase Auth. ستتولى الـ Triggers إنشاء بيانات الـ Profiles بشكل آلي. يجب عليك لاحقاً تغيير `role` لمدير واحد على الأقل إلى `admin` يدوياً من داخل الجدول للبدء.

3. أضف متغيرات البيئة:
   قم بإنشاء ملف `.env.local` أو انسخ محتوى `.env.example` وأضف بياناتك:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. شغّل خادم التطوير:
   ```bash
   npm run dev
   ```
